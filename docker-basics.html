1. Docker Install kaise kare?

Ubuntu/Linux par Docker install karne ke liye:

sudo apt update
sudo apt install docker.io -y
sudo systemctl start docker
sudo systemctl enable docker
docker --version

Windows ya Mac par aap Docker Desktop install kar sakte ho.

2. Docker Image kaise banate hai?

Ek Dockerfile banaiye:

# Dockerfile
FROM ubuntu:20.04
RUN apt-get update && apt-get install -y python3
CMD ["python3", "--version"]

üëâ Image build karne ke liye command:

docker build -t mypython:1.0 .

Fir container run kijiye:

docker run mypython:1.0
