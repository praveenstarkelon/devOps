Introduction

Docker ek open-source platform hai jo application ko container ke andar run karta hai. Ye lightweight, portable aur repeatable environments deta hai.

Installation

Docker ko install karne ke liye aap Windows, macOS, ya Linux ke liye official documentation follow kar sakte ho.

Docker Images

Images ek read-only template hota hai jisme application aur uske dependencies bundled hote hain. Example: Ubuntu, Nginx images.

Docker Containers

Container ek running instance hota hai ek image ka. Ye isolate hota hai system se aur easily start/stop kiya ja sakta hai.

Volumes

Volumes ka use data ko persist karne ke liye hota hai, taki container delete hone par bhi data safe rahe.

Networking

Docker networking se containers ek dusre ke sath communicate kar sakte hain aur external world se connect ho sakte hain.
