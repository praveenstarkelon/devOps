Kubectl installation

ğŸ”¹ Method 1: Direct Binary Download (Recommended)

Isme hum kubectl.exe ko download karke PATH me add karenge.

Step 1: Latest Version Check Kare

Kubernetes ka latest stable version check karne ke liye browser me ye URL khole:
https://dl.k8s.io/release/stable.txt

Example: v1.34.0

Step 2: kubectl.exe Download Kare

Command Prompt (cmd) ya PowerShell open karke run kare:

 
curl.exe -LO "https://dl.k8s.io/release/v1.34.0/bin/windows/amd64/kubectl.exe"

 Agar aapke system me curl nahi hai, to browser se manually download kare aur kubectl.exe ko save kare C:\kubectl\ folder me.

Step 3: PATH Variable me Add Kare

  1. Win + R dabaye → sysdm.cpl type kare → Enter.

  2. Advanced tab → Environment Variables.

  3. "System variables" section me Path select kare → Edit.

  4. New dabaye aur jaha aapne kubectl.exe rakha hai uska path add kare. Example:

     
    C:\kubectl

  5. OK → OK → OK dabakar sab close kare.

Step 4: Installation Verify Kare

New cmd/PowerShell khole aur run kare:

 
kubectl version --client

Output me Client Version: v1.34.0 (ya jo bhi version hai) aana chahiye. âœ…

Delete karne k liye binary delete kar do , and path remove kar do

ğŸ”¹ Method 1: Direct Binary Download (Official Kubernetes Way)


Step 1: Latest Stable Version Check Kare
curl -L -s https://dl.k8s.io/release/stable.txt


Example output: v1.34.0

Step 2: Kubectl Binary Download Kare
curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl"

Step 3: Binary Ko Executable Bana Do

Step 4: Move to PATH
sudo mv kubectl /usr/local/bin/

Step 5: Verify Installation
kubectl version --client

ğŸ”¹ Method 2: Install via Package Manager

Ubuntu / Debian (apt)

 
sudo apt-get update 
 
sudo apt-get install -y apt-transport-https ca-certificates curl
 
sudo curl -fsSLo /usr/share/keyrings/kubernetes-archive-keyring.gpg https://dl.k8s.io/apt/doc/apt-key.gpg
 
echo "deb [signed-by=/usr/share/keyrings/kubernetes-archive-keyring.gpg] https://apt.kubernetes.io/ kubernetes-xenial main" | sudo tee /etc/apt/sources.list.d/kubernetes.list
 
sudo apt-get update
 
sudo apt-get install -y kubectl
 
 
 
 
 
 

 

Remove (Uninstall):

  • Manual binary: sudo rm /usr/local/bin/kubectl

  • Apt: sudo apt-get remove kubectl
